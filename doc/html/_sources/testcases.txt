#summary Influences on other Projects

= Influences on other Projects =

 * The Python interface to the low-level FUSE library that has been developed for S3QL has been added to [http://code.google.com/p/fusepy/ FusePY], so that it can be used by other FUSE file systems as well.

 * S3QL development has helped to reveal some bugs in other open source projects. So far, these are:
   * [http://bugs.python.org/issue7736 Bug 7736] in Python
   * [https://bugs.launchpad.net/pyliblzma/+bug/521975 Bug 7736] in PyLibLZMA
   * [http://bugs.python.org/issue6627 Bug 6627] in Python
   * [http://www.sqlite.org/cvstrac/tktview?tn=3992 Bug 3992] in SQLite
