GCC-XML needs to be compiled from the CVS sources: {{{ cvs -d :pserver:anoncvs@www.gccxml.org:/cvsroot/GCC_XML login #(just press enter when prompted for a password) cvs -d :pserver:anoncvs@www.gccxml.org:/cvsroot/GCC_XML co gccxml #(this is the slowest step of the whole tutorial. Go get a cup of mocha) mkdir gccxml-build cd gccxml-build cmake ../gccxml make make test make install }}}

PyCryptoPP: {{{ wget http://pypi.python.org/packages/source/p/pycryptopp/pycryptopp-0.5.17.tar.gz#md5=b3d19e7203531f8bd241ae58062f99e4 tar -xf pycryptopp-0.5.17.tar.gz cd pycryptopp* python26 setup.py build python26 setup.py test python26 setup.py install }}}

For OpenVZ, execute in host the below: {{{ modprobe fuse vzctl set 101 –devices c:10:229:rw –save vzctl exec 101 mknod /dev/fuse c 10 229 vzctl exec 101 reboot }}}
