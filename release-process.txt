For making a new release:

 - Bump version in src/s3ql/__init__.py
 - Add release date to Changes.txt
 - Bump version in debian/changelog
 - hg commit, hg tag

 - Build & upload documentation
 - Build & upload source distribution
 - Deprecate old source distribution

 - Check for new apsw version and copy into PPA if so
 - Check for new pyliblzma version and update PPA
   if required
 - Check for new FUSE version and update PPA if required

 - Upload Lucid version to launchpad:
 - Write announcement to mailing list
